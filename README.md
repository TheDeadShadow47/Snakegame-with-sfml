# Classic Snake (SFML 3.0)

A simple, grid-based Snake game built using **C++** and **SFML 3.0**. This was developed as a learning project to master modern C++ and graphics library integration.

---

## Gameplay
* **Objective:** Eat the red fruit to grow your snake.
* **Avoid:** Don't run into your own body!
* **Wrapping:** The screen is wrap-around; hitting a wall sends you to the opposite side.

### Controls
* **Arrow Keys:** Move Up, Down, Left, or Right.
* **Window Close:** Exit the game.

---

## Requirements
This project uses **SFML 3.0.2**. To build it yourself, you need:

1.  **Compiler:** GCC 14.2.0 (MinGW-w64 UCRT) or similar.
2.  **Library:** [SFML 3.0+](https://www.sfml-dev.org/download.php) 
    * *Note: Ensure your SFML version matches your compiler (UCRT vs MSVCRT).*
